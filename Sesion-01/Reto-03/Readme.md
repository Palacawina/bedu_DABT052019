## Creando un Script con Vim

### OBJETIVO 
 - Crear y editar archivos básicos con Vim.

#### REQUISITOS 
1. Vim
2. Consola 

### DESARROLLO
Crea un script que automáticamente crea la siguiente estructura de proyecto
```
mi_analisis
    └── README.md
    └── datos
    └── codigo
        └── analisis.py
        └── tests.py
        └── experimentacion
            └── borrar.py
```
